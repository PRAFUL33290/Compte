🎮 CONCEPT — RECONSTRUIRE LES HISTOIRES
🧠 Objectif

L’enfant reconstruit une histoire dans le bon ordre sur une grille.
Chaque histoire suit : début → défi 🔥 → solution → fin
👉 Apprentissage logique (cause → conséquence)

🕹️ Gameplay
Sélection + glisser = placement continu
Reconstruction de l’histoire
Feedback immédiat :
✅ réussite → animation + son
❌ erreur → retour visuel
📚 Histoires (résumé)

10 univers variés avec problèmes à résoudre :

🐰 Lapin perdu → traverse un ruisseau
🚀 Espace → réparer une fusée
🏴‍☠️ Pirate → tempête + coffre fermé
🐉 Dragon → éteindre un incendie
🚒 Pompier → route bloquée + manque d’eau
🦖 Dinosaure → sortir d’un trou
🚗 Voiture magique → survivre à une tempête
🐠 Océan → se libérer d’un filet
🌈 Arc-en-ciel → restaurer les couleurs
🍰 Gâteau → trouver les ingrédients
🧩 Assets à créer
🌿 Environnements

Forêt, île, château, jungle, océan, ciel, monde arc-en-ciel

🧱 Défis

Pont cassé, tempête, coffre fermé, feu, obstacle, trou, filet, monde gris, panne

🛠️ Solutions

Branches, pièces, clé, planche, borne incendie, liane, cristal, couleurs, ingrédients

👤 Personnages

Lapin, enfant, alien, pirate, princesse, dragon, pompier, dinosaure, requin, amis

💡 Améliorations clés
Scènes pièges pour challenger
Difficulté progressive (3 → 8 scènes)
Système d’étoiles ⭐
Animation finale récompense
