1. Le Lapin Perdu et le Ruisseau

Début : Le petit lapin est dans la forêt et veut aller voir sa grand-mère.
→ Forêt + Lapin
Défi 🔥 : Il arrive devant un ruisseau avec un pont cassé.
→ Pont cassé + Ruisseau
Solution : Il ramasse des branches pour construire un nouveau pont.
→ Branches
Fin : Le lapin traverse et fait un gros câlin à sa grand-mère !
→ Grand-mère (nouvel asset simple : lapin + cœur)


2. L’Astronaute et la Fusée Cassée

Début : L’alien est dans son vaisseau spatial, prêt à explorer l’espace.
→ Ciel + Alien
Défi 🔥 : La fusée tombe en panne et ne peut plus voler.
→ Panne + Fusée cassée
Solution : Il trouve les bonnes pièces et les remet en place.
→ Pièces
Fin : La fusée repart dans les étoiles avec des lumières qui brillent !
→ Fusée qui vole (animation étoiles)


3. Le Pirate et la Tempête

Début : Le pirate est sur son île avec son bateau.
→ Île + Pirate
Défi 🔥 : Une grosse tempête arrive et un coffre magique est fermé à clé.
→ Tempête + Coffre fermé
Solution : Il utilise une clé trouvée dans la jungle pour ouvrir le coffre.
→ Clé
Fin : Le pirate trouve le trésor et danse de joie sous le soleil.
→ Trésor + Pirate joyeux


4. Le Dragon et l’Incendie

Début : Le gentil dragon vit dans son château.
→ Château + Dragon
Défi 🔥 : Un feu commence dans la forêt près du château.
→ Feu + Forêt
Solution : Il va chercher la borne incendie et éteint les flammes.
→ Borne incendie
Fin : Tout le monde est en sécurité et le dragon fait un grand sourire.
→ Château + Dragon content


5. Le Pompier Courageux

Début : Le pompier est dans sa caserne et tout va bien.
→ Château (ou ville) + Pompier
Défi 🔥 : La route est bloquée et il n’y a plus d’eau pour éteindre le feu.
→ Obstacle + Manque d’eau
Solution : Il utilise une planche pour dégager la route et remplit le camion.
→ Planche + Borne incendie
Fin : Il sauve la maison et tout le monde applaudit !
→ Maison sauvée + Pompier fier


6. Le Dinosaure dans le Trou

Début : Le dinosaure se promène joyeusement dans la jungle.
→ Jungle + Dinosaure
Défi 🔥 : Il tombe dans un grand trou et ne peut plus sortir.
→ Trou
Solution : Il attrape une longue liane pour grimper et sortir.
→ Liane
Fin : Le dinosaure est libre et retourne jouer avec ses amis.
→ Dinosaure + Amis


7. La Voiture Magique et la Tempête

Début : La voiture magique roule tranquillement sur la route.
→ Forêt ou Île + Voiture magique
Défi 🔥 : Une grosse tempête arrive et bloque le chemin.
→ Tempête + Obstacle
Solution : Elle utilise ses pouvoirs magiques + une planche pour passer.
→ Planche + Étoiles magiques (nouvel asset simple)
Fin : La voiture arrive à la fête avec tous ses amis.
→ Voiture + Amis


8. Le Requin et le Filet

Début : Le petit requin nage librement dans l’océan.
→ Océan + Requin
Défi 🔥 : Il se retrouve coincé dans un grand filet de pêche.
→ Filet
Solution : Ses amis poissons lui apportent un cristal tranchant pour couper le filet.
→ Cristal
Fin : Le requin nage à toute vitesse et fait des loopings de joie !
→ Océan + Requin joyeux


9. L’Arc-en-Ciel Gris

Début : La princesse se promène dans le monde arc-en-ciel.
→ Monde arc-en-ciel + Princesse
Défi 🔥 : Tout devient gris, les couleurs ont disparu !
→ Monde gris
Solution : Elle trouve les cristaux de couleurs et les remet en place.
→ Cristal + Couleurs
Fin : L’arc-en-ciel brille à nouveau et tout le monde danse.
→ Arc-en-ciel lumineux


10. Le Grand Gâteau Magique

Début : L’enfant est dans la cuisine et veut faire un énorme gâteau.
→ Château (cuisine) + Enfant
Défi 🔥 : Il manque tous les ingrédients pour terminer le gâteau.
→ Manque d’ingrédients
Solution : Il part chercher les ingrédients dans la forêt et la jungle.
→ Ingrédients
Fin : Le gâteau est prêt, tout le monde mange et fait la fête !
→ Gâteau + Amis
